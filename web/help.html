Open Pipeline Help

Open Pipeline is a tool designed to help people with production pipelines, when using open source software. Open Pipeline is licensed under the GPL3 license, granting you the right to do many things.

Open Pipeline supports multiple Open Source software out of the box, including OpenToonz and Blender. To connect Open Pipeline to the specified software, you must download the required plugins from the Open Pipeline github repository, and install them into the software of choice.
